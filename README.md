## Install

```sh
npm install
```

## Env variables

- MONGO_USER : Mongodb user
- MONGO_USER_PWD : Mongodb password
- MONGO_NAMESPACE : Mongodb namespace
- SECRET

## Usage (with node)

> start the api

```sh
npm run start
```

> start the client

```sh
cd client
npm run start
```
